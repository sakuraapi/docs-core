cipherKey: function

Returns the cipher key to be used by @Json decorated properties with [[IJsonOption.encrypt]] === true. Provide a function that returns the key as a string. The function will be given the context of this of the model from which it is being called.

Optional dbConfig

dbConfig: object
  • Defined in core/@model/model.ts:76

The database to which this @Model object is bound

Type declaration

  • Optional collation?: IMongoDBCollation

    Optionally set the default MongoDB collation for this model. For valid settings see: https://docs.mongodb.com/manual/reference/collation-locales-defaults/#collation-languages-locales

  • collection: string

    The name of the collection in the database referenced in db that represents this model.

  • db: string

    The name of the database for this model which is used to retrieve the MongoDB Db object from SakuraMongoDbConnection. I.e, whatever name you used in your congifuration of SakuraMongoDbConnection for the database connection for this model, use that name here.

  • Optional promiscuous?: boolean

    If true, fields without an Explicit @Db will still be written to the Db and used to rehydrate objects fromDb.
